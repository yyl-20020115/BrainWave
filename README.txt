人的四种基本脑波包括δ波、θ波、α波和β波。每种脑波代表着不同的脑活动状态，对我们的认知、
情绪和行为具有重要影响。

1、δ波：是一种低频脑电波，频率在0.5-4赫兹之间。主要在深度睡眠状态下出现，是大脑休息和修复的标志，
可帮助身体恢复、记忆巩固和维持免疫系统的正常功能。

2、θ波：频率在4-8赫兹之间，主要在睡眠、催眠和放松状态下出现，多与创造性思维、直觉和潜意识的表达有关。
一些冥想和放松技术也可以促进θ波的产生，帮助放松身心、提升注意力和集中力。

3、α波：频率在8-13赫兹之间，主要在放松但清醒的状态下出现。这种脑波与放松、注意力调节和学习记忆有关，
是一种平静而稳定的状态。

4、β波：频率在13-30赫兹之间，主要在清醒、专注和思考的状态下出现。
一般与大脑的激活状态、思维加工和工作记忆有关。

每个人的脑波模式可能存在差异，且受到环境、心理状态和健康状况的影响。如果对脑波方面有特定疑虑，
建议咨询专业人员。
